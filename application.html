One Application
with unlimited possibilities

Via integration with the World Wind environment, this client can provide relevant geospatial web service standards including WMS, WFS, and WCS (e.g., via getting information and capabilities, rendering visualizations, providing data results), focused on marine traffic and climate oriented datasets. The client includes three major modules: data management, visualization module, and analytical module. 
The overall design of the client GUI is user friendly yet powerful. This interface component enables user manipulation of datasets, classification and analysis functions, and information and visualization outputs. Users, in addition, are provided with standard zoom, pan, adjusting, and data locating tools, all in two and three dimensions.
